# Fistat

Staattinen GitHub Pages -yhteensopiva HTML-työkalu Tilastokeskuksen rikos- ja pakkokeinotilaston PxWeb-aineistojen selaamiseen.

## Käyttö

Avaa `index.html` selaimessa tai julkaise repositorio GitHub Pages -sivuna. Työkalu hakee metatiedot ja datan suoraan Tilastokeskuksen avoimesta PxWeb APIsta käyttäjän selaimessa.

## Lähteet

- Tilastokeskus: Rikos- ja pakkokeinotilasto / StatFin PxWeb
- Tilastokeskus: tietokantojen rajapintakäyttö ja PxWeb API

## Huomioita

Tilastot ovat aggregoituja viranomaistilastoja. Tulkinnassa pitää huomioida väestöön suhteutetut luvut, ikä- ja sukupuolirakenne, ilmoitusalttius, rikoslakimuutokset ja pienet havaintomäärät.
